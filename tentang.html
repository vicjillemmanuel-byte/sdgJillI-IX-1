Refleksi Pengerjaan tugas



Dari tugas ini, saya belajar banyak mengenai SDG. Saya tidak pernah tahu bahwa ternyata pemerintah memiliki agenda 2030, dan menurut saya belajar mengenai tujuan-tujuan ini sangatlah seru. Melalui proses mencari informasi, saya jadi semakin memahami bagaimana berbagai negara bekerja sama untuk mengatasi masalah global seperti kemiskinan, ketimpangan, pendidikan, dan lingkungan. Saya juga baru menyadari bahwa banyak program di Indonesia ternyata terhubung dengan SDG, meskipun tidak dikatakan secara langsung. Belajar tentang SDG membuat saya melihat masalah masalah yang dihadapi semua negara sekarang ini,

Saya melihat ketimpangan di sekitar saya. Contohnya, saat pergi ke sekolah. Ada jalan yang terlihat sangat terawat, tapi mungkin beberapa km kemudian, jalannya bolong, rusak dan kotor. Ternyata mengurangi ketimpangan masuk ke dalam SDG ke 10. Saya jadi lebih peka terhadap isu ketidakadilan setelah mengerjakan tugas ini.

Victoria Jill Emmmanuel
